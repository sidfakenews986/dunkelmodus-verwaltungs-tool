# Dunkelmodus Verwaltungs Tool

## Überblick
Dieses Projekt bietet Entwicklern ein leistungsstarkes Tool, um den Dunkelmodus in ihren Webanwendungen einfach zu integrieren und zu verwalten. Es ermöglicht Benutzern, zwischen Hell- und Dunkelmodus umzuschalten und Anpassungen an den Farben und Stilen vorzunehmen, um eine benutzerfreundliche Erfahrung zu gewährleisten.

## Funktionen
- **Einfache Integration**: Fügen Sie das Tool in Ihrer Webanwendung mit nur wenigen Zeilen Code hinzu.
- **Benutzerdefinierte Farben**: Passen Sie die Farben für den Dunkelmodus an, um ein einzigartiges Benutzererlebnis zu schaffen.
- **Speicherung der bevorzugten Einstellung**: Speichern Sie die Dunkelmodus-Einstellung für jeden Benutzer mithilfe von Cookies oder localStorage.
- **Responsive Design**: Das Tool funktioniert auf allen Bildschirmgrößen und Geräten.

## Installation
1. Klonen Sie dieses Repository:
   ```bash
   git clone https://github.com/IhrBenutzername/dunkelmodus-verwaltungs-tool.git
   ```
2. Installieren Sie die Abhängigkeiten:
   ```bash
   npm install
   ```
3. Fügen Sie das Tool in Ihre Anwendung ein:
   ```javascript
   import { DarkModeToggle } from 'dunkelmodus-verwaltungs-tool';
   ```

## Nutzung
Verwenden Sie die bereitgestellten Funktionen, um den Dunkelmodus in Ihrer Anwendung anzupassen und zu steuern.
